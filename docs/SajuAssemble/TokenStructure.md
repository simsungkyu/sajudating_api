# 1페이지 요약: 카테고리 & 위치 표준 토큰 세트

## A) 토큰 기본 문법(표준)

```
<카테고리>:<이름>[@<위치>][#<등급>]
```

- `@위치`와 `#등급`은 필요할 때만 붙임
- 등급은 **L/M/H** (w=0~100을 엔진에서 등급으로 변환)

**예시**

| 구분   | 예시                          |
|--------|-------------------------------|
| 존재   | `십성:정재`                   |
| 위치   | `십성:정재@월간`              |
| 등급   | `십성:정재#H`                 |
| 위치+등급 | `십성:정재@월간#H`         |
| 이벤트 | `관계:충@일지-년지#H`         |

---

## B) 카테고리 목록(표준 9개)

아래 9개로 고정하면 대부분 커버됨.

| 카테고리 | 설명 | 예시 |
|----------|------|------|
| **십성** | 이름 예: 비견/겁재/식신/상관/편재/정재/편관/정관/편인/정인 | `십성:편관@년간#M` |
| **관계** | 합/충/형/파/해 + 확장 이벤트. 이름 예: 합/충/형/파/해/천간합/삼합/방합/반합 | `관계:충@일지-년지#H`, `관계:천간합@일간-월간` |
| **오행** | 이름 예: 목/화/토/금/수 | `오행:토#H`, `오행:수#L` |
| **신살** | 이름 예: 도화/역마/천을귀인…(룰셋에 따라 확장) | `신살:도화@일지` |
| **격국** | 이름 예: 정재격/정관격/식신격…(+ 네가 쓰는 체계) | `격국:정재격` (등급이 필요하면 `#H`) |
| **용신** | 이름 예: 목/화/토/금/수 (또는 "용신:수") | `용신:수` (확신은 `확신:용신#M`로 분리 추천) |
| **육친** | 이름 예: 부친/모친/배우자/자녀/형제자매 등(정의한 세트) | `육친:부친#H` (강조도는 등급으로) |
| **강약** | 이름 예: 신강/중화/신약 (또는 strong/neutral/weak) | `강약:중화`, `강약:신강#M` |
| **확신** | confidence / quality / reliability. 이름 예: 십성/신살/용신/격국/전체 | `확신:전체#H`, `확신:용신#L` |

> **팁:** 유파가 갈리는 건 확신으로 "말투/단정"을 조절하는 게 운영에 편함.

## C) 위치 표준 토큰 세트(원국 기준)

### 1) 기본 8개 위치(가장 많이 씀)

- 년간 / 년지
- 월간 / 월지
- 일간 / 일지
- 시간 / 시지

**예:**

- `십성:정재@월간`
- `신살:도화@일지`

### 2) 지장간 위치(필요 시)

- 년지.지장간 / 월지.지장간 / 일지.지장간 / 시지.지장간

**예:**

- `십성:정재@월지.지장간`

### 3) 관계(이벤트) 위치 표준

관계는 "두 지점 사이"로 표기:

- `@<A>-<B>` 형태로 고정 (순서는 고정 규칙 추천: 년→월→일→시)

**예:**

- `관계:충@일지-년지`
- `관계:천간합@일간-월간`

### 4) (선택) 전체/요약 위치

- `@전체` : "원국 전체에서"라는 의미로 사용할 수 있음  
  예: `오행:토@전체#H`, `확신:전체#M`

---

## D) 등급(L/M/H) 권장 적용 대상

- **적용 대상:** 십성 / 오행 / 관계 / 강약 / 확신 / (격국·용신은 선택)
- **신살**은 보통 존재+위치만으로 충분 (`#`은 선택)

**등급 기준(예시)**

| 등급 | 범위   |
|------|--------|
| L    | 0~49   |
| M    | 50~69  |
| H    | 70~100 |

(기준은 엔진에서 통일하고, 카드에는 숫자 쓰지 않기 추천)

---

## E) 카드 조건 비교(한 줄)

카드의 trigger는 위 토큰들을 **Set 조회**로 검사:

| 연산 | 의미               |
|------|--------------------|
| **all** | 전부 있어야 함   |
| **any** | 하나라도 있으면 됨 |
| **not** | 하나라도 있으면 탈락 |
